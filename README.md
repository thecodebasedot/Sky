# Sky вҳҒпёҸ

A cross-platform **mobile communication app** for Android and iOS, built with
[Flutter](https://flutter.dev) from a single codebase.

Sky is being built toward full messaging, voice & video calls, media sharing,
and status updates. This repository holds all of the engineering work.

### рҹ”‘ Trying the sign-in flow
The app boots to a welcome screen. Tap **Agree & continue**, enter any phone
number, then on the verification screen enter **`123456`** (or any 6 digits).
Auth runs against an in-memory `MockAuthService`; nothing is sent or stored.

> The app currently runs entirely on local **mock data** so the full UI is
> browsable before any backend exists. State lives in `ChatStore`, which is
> deliberately backend-agnostic вҖ” swap its methods for real API calls later.

## рҹҡҖ Getting started

### Prerequisites
- [Flutter SDK](https://docs.flutter.dev/get-started/install) 3.27 or newer
- Android Studio / Xcode for device emulators

### First-time setup
The native **Android** and **iOS** project shells are generated by Flutter and
are not checked in. Run this once in the repo root to create them (it preserves
all existing `lib/` code):

```bash
flutter create .
```

### Install dependencies & run
```bash
flutter pub get
flutter run            # runs on a connected device / emulator
```

### Build release binaries
```bash
flutter build apk      # Android
flutter build ios      # iOS (requires macOS + Xcode)
```

### Run tests
```bash
flutter test
```
